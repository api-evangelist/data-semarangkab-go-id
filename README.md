# Satu Data Semarang Regency (data-semarangkab-go-id)

Satu Data Semarang Regency is a regional government open-data portal for Indonesia running **CKAN**, exposing a standardized catalog API over ~467 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-semarangkab-go-id/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-semarangkab-go-id-api-evangelist&utm_content=repo)
